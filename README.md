# Vendor Portal

A modern vendor management portal built with React and Vite.

## Features

- Vendor Dashboard with analytics
- Product management
- Order tracking
- Inventory management
- Wallet and payments
- Profile management
- Real-time notifications

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Heroicons
- React Router
- Recharts
- Material-UI
